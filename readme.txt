sassのコンパイルと、画像の圧縮、それ以外のファイルのコピーと
ローカルホストの立ち上げおよび自動更新を行います。

編集ファイルはsrcフォルダ内で作業
出力ファイルはpublic_htmlフォルダに出力されます。
出力ファイルにはsourcemapも含まれるので、
どのファイルに書いたscssがきいているのかは分かりやすくなるかと思います

出力結果はgulpでlocalhostを立ち上げ自動で更新されます
（規定のブラウザで開きますが、
gulpを動かした状態でURLを違うブラウザに貼り付けて表示させると
そのブラウザでも同様に自動更新が行われます。）

gulpfile.js：gulpの処理を制御
package.json：インストールさせるパッケージの一覧
（適切に作成されていればnpm installだけでインストールが終了する）
.editorconfig：各エディタの基本設定を共有する（タブにするかスペースにするかスペースなら4つか2つかなど）
ただし事前に各エディタにプラグインのインストールが必要


使用手順
（gulpのグローバルインストールは完了しているものとする）

編集フォルダに一式ファイルをコピー
PowerShellやcmd.exeなどのコマンドプロンプトを立ち上げ、
gulpfile.jsのあるフォルダに移動する
例）cd 飛び先

npm installを実行する

gulpを実行する
（gulpコマンドに監視機能も含めているため改めて監視用コマンドを入力する必要はない）
